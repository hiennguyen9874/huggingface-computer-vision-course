07 · Các đơn vị cơ bản

Pixel, voxel, channel, mask.

  • Pixel — phần tử nhỏ nhất của ảnh 2D.
  • Voxel — phần tử nhỏ nhất của ảnh 3D: MRI, CT, volume.
  • Channel — lớp thông tin: RGB, infrared, thermal, X-ray.
  • Mask — ảnh nhị phân: 1 = foreground, 0 = background.

11 · Preprocessing

Input phải nhất quán trước khi học.

preprocess.py
img = cv2.imread("image.jpg")
resized = cv2.resize(img, (224, 224))
blurred = cv2.GaussianBlur(resized, (5, 5), 0)
gray = cv2.cvtColor(blurred, cv2.COLOR_BGR2GRAY)
edges = cv2.Canny(gray, 100, 200)

# neural network input
x = img.astype("float32") / 255.0

13 · Khôi phục ảnh

Enhancement, restoration hay reconstruction?

Khái niệmMục tiêuVí dụ
Enhancement

Cải thiện ảnh theo mục tiêu thị giác; thường chủ quan.

brightness · contrast · sharpen · histogram equalization

Restoration

Khôi phục ảnh dựa trên mô hình suy giảm.

denoise · deblur · remove artifacts · ảnh cũ

Reconstruction

Tạo ảnh từ dữ liệu đo gián tiếp.

CT projections · MRI · 3D từ nhiều ảnh 2D

14 · Color

Màu là nhiều kênh thông tin.

RGB 8-bit có 256³ = 16.777.216 màu. CMY/CMYK phục vụ in ấn. Pseudo-color gán màu giả để đọc ảnh nhiệt hoặc grayscale dễ hơn.

15 · Compression

Nén bằng cách tận dụng redundancy.

Coding

Giá trị phổ biến → code ngắn hơn. Huffman.

Spatial

Pixel lân cận giống nhau. Run-length.

Temporal

Frame sau gần frame trước. Video lưu phần khác biệt.

Lossy

Loại thông tin ít quan trọng. JPEG.

25 · Mini pipeline thực tế

Train và validation không nên dùng cùng một chiến lược.

transforms.py
train_transform = T.Compose([
  T.Resize((256, 256)),
  T.RandomResizedCrop(224), T.RandomHorizontalFlip(),
  T.ColorJitter(brightness=0.2, contrast=0.2),
  T.ToTensor(), T.Normalize(mean, std)
])
val_transform = T.Compose([
  T.Resize((224, 224)),
  T.ToTensor(), T.Normalize(mean, std)
])

Train: augmentation ngẫu nhiên · Validation: ổn định · Normalize giống nhau ở inference.
