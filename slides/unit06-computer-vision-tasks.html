Hugging Face pipeline

Classification trong vài dòng.

classify.py 
from transformers import pipeline
from PIL import Image

classifier = pipeline(
    "image-classification",
    model="google/vit-base-patch16-224"

print(result)  # label + score

Output thường là danh sách nhãn kèm score (thường là xác suất): golden retriever · 0.91

Semantic Segmentation

Hiểu “vùng” của scene.

  • Road, sky, tree, house, water.
  • Phù hợp với ảnh vệ tinh, đất nông nghiệp, ảnh y tế.
  • Không phân biệt từng object cùng class.

Instance Segmentation

Hiểu “từng vật thể”.

  • Đếm số lượng vật thể.
  • Robot grasping, tách tế bào, tách từng người.
  • Mỗi instance có label, score, mask riêng.

Panoptic Segmentation

Một scene hoàn chỉnh cần cả “stuff” lẫn “things”.

  • Stuff — vùng không cần đếm instance: road, sky, grass, wall, water.
  • Things — vật thể đếm được: person, car, dog, chair, bicycle.
  • Panoptic trả lời đồng thời: pixel này thuộc class nào và instance nào nếu có?

Promptable segmentation

SAM: đưa ảnh vào, nhận các mask ứng viên.

sam.py 
from transformers import pipeline
from PIL import Image

pipe = pipeline(
    "mask-generation",
    model="facebook/sam-vit-base",
    device=0  # bỏ nếu chạy CPU
)
raw_image = Image.open("image.jpg").convert("RGB")
masks = pipe(raw_image)
print(masks)  # nhiều mask ứng viên

Ví dụ zero-shot: có thể áp dụng lên ảnh chưa thấy mà không cần fine-tune trực tiếp; chất lượng vẫn phụ thuộc prompt và domain.

Đánh giá Segmentation

Mask tốt phải thắng cả overlap lẫn imbalance.

Metric Công thức Đo điều gì? Lưu ý
IoU / Jaccard intersection / union Overlap giữa predicted mask và ground truth Ít bị ảnh hưởng hơn bởi class imbalance.

Dice Coefficient 2 × intersection / (pred + true) Độ chồng lấp nhạy với vùng nhỏ Rất hay dùng trong medical imaging.

Metric trong NumPy

Cùng một ý tưởng: giao, hợp, diện tích.

metrics.py 
import numpy as np

def mask_iou(pred, true):
    pred, true = pred.astype(bool), true.astype(bool)
    inter = np.logical_and(pred, true).sum()
    union = np.logical_or(pred, true).sum()
    return inter / union if union else 1.0

def pixel_accuracy(pred, true):
    return (pred == true).sum() / true.size

def dice(pred, true):
    pred, true = pred.astype(bool), true.astype(bool)
    inter = np.logical_and(pred, true).sum()
    total = pred.sum() + true.sum()
    return 2 * inter / total if total else 1.0

Ví dụ dùng mask nhị phân; nhiều class cần tính từng class rồi average. Nếu union hoặc tổng diện tích bằng 0, cần quy ước rõ output.
