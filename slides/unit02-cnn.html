    Tại sao 1×1?

    Bottleneck trước convolution lớn

    5×5 trực tiếp: 819,200 params

    1×1: 128→64, rồi 5×5: 417,792 params

    Ngoài giảm channel, 1×1 còn tăng channel, thêm phi tuyến và thay FC bằng average pooling.

    Auxiliary classifier: hỗ trợ gradient ở mạng sâu khi train; bỏ khi inference.

    ResNet details

    Khi tensor không cùng shape

    • Zero-padding shortcut: thêm channel bằng zero, không thêm tham số học.
    • Projection shortcut: dùng `1×1 conv` để đổi channel hoặc spatial size.
    • Bottleneck: `1×1 giảm → 3×3 xử lý → 1×1 tăng`; dùng trong ResNet-50/101/152.
    • Input pretrained thường cần RGB, `3×H×W`, size khoảng 224+, normalize theo ImageNet.

    Degradation problem

    deeper ≠ better

    Mạng sâu hơn có thể có training error cao hơn — không nhất thiết do overfitting, mà do tối ưu khó.

    Depthwise separable convolution

    Tách spatial mixing khỏi channel mixing

    Implementation

    Depthwise → pointwise → normalize → activate

    mobile_block.py
    class DepthwiseSeparableConv(nn.Module):
  def __init__(self, cin, cout, stride=1):
    super().__init__()
    self.depthwise = nn.Conv2d(cin, cin, 3, stride,
                                1, groups=cin, bias=False)
    self.pointwise = nn.Conv2d(cin, cout, 1, bias=False)
    self.bn = nn.BatchNorm2d(cout)
    self.relu = nn.ReLU(inplace=True)
  def forward(self, x):
    return self.relu(self.bn(self.pointwise(self.depthwise(x))))

    Vai trò của 1×1 conv

    Không nhìn hàng xóm — nhưng hiểu channel

    • Trộn thông tin giữa các channel tại từng pixel.
    • Giảm/tăng channel, tạo bottleneck và giảm chi phí.
    • MobileNet phù hợp điện thoại, edge device và real-time budget.
    • Có thể ghép MobileNet với Transformer: local features + global context, ví dụ Mobile-Former.

    MobileNet goal

    fast ∧ small

    Đánh đổi một phần accuracy để giảm latency, memory và FLOPs.

    YOLO objective

    Loss cần cân bằng giữa box, objectness và class

    • Localization: phạt sai tọa độ; `λcoord = 5`.
    • No-object confidence: rất nhiều cell rỗng; giảm trọng số với `λnoobj = 0.5`.
    • Width/height: dùng `sqrt(w), sqrt(h)` để box nhỏ được coi trọng hơn.
    • Class: `P(classᵢ | Object) × confidence`.
