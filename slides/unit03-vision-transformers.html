CNN

Bias hữu ích cho ảnh

  • Locality: pixel gần nhau thường liên quan.
  • Translational equivariance: filter trượt trên toàn ảnh.
  • Thường mạnh khi dataset nhỏ.

ViT

Ít giả định hơn

  • Cần dữ liệu và compute lớn hơn để học đặc trưng.
  • Scale tốt khi có pretraining lớn.
  • Thực tế: dùng pretrained rồi fine-tune.

Decision map

DETR mechanics

Không anchor. Không NMS.

  • Dự đoán một số lượng cố định object queries, ví dụ 100.
  • Mỗi query trả về object hoặc class đặc biệt no object.
  • Hungarian algorithm match prediction với ground truth.
  • Loss: classification + bounding box + generalized IoU.

Set-based: [cat, dog] và [dog, cat] là cùng một tập.

Q01person · box
Q02helmet · box
Q03no-object
Q04no-object

DETR inference

Post-process box về kích thước ảnh

detr_infer.py
processor = DetrImageProcessor.from_pretrained(checkpoint)
model = DetrForObjectDetection.from_pretrained(checkpoint)
inputs = processor(images=image, return_tensors="pt")
outputs = model(**inputs)

    outputs, target_sizes=target_sizes, threshold=0.9
)[0]
for score, label, box in zip(results["scores"], results["labels"], results["boxes"]): print(label, score, box)

ViT gốc

Global attention

N patch → mọi patch nhìn mọi patch. Chi phí attention là O(N²), ảnh lớn trở nên đắt.

Window + shift

Attention cục bộ trong window nhỏ; layer sau shift window để patch ở vùng khác giao tiếp.

window attention → shifted window → cross-window context

Swin family

Hierarchical như CNN

  • Patch merging: gom 2×2 patch → H,W giảm một nửa; số channel thường tăng gấp đôi.
  • Các stage tạo feature map đa scale cho classification, detection, segmentation.
  • Swin V2: ổn định training model lớn, hỗ trợ resolution cao, dùng SimMIM.
  • SwinIR / Swin2SR: image restoration, nổi bật là super-resolution.

Inductive bias

Đưa locality trở lại Transformer

CvT · DiNAT · MobileViT v2

CvT

Convolutional Vision Transformer

  • Convolutional token embedding: receptive field cục bộ, token overlap, downsample tự nhiên.
  • Convolutional projection: Q, K, V dùng depthwise separable convolution.
  • Convolution đã encode không gian → bỏ positional embedding.
  • Có tiềm năng hỗ trợ input resolution linh hoạt hơn, ít phụ thuộc fixed size.

Knowledge distillation

CE giữ đúng nhãn, KL giữ quan hệ

distill_loss.py
def distillation_loss(student, teacher, labels, T=2.0, alpha=0.5):
    ce = F.cross_entropy(student, labels)
    student_soft = F.log_softmax(student / T, dim=-1)
    teacher_soft = F.softmax(teacher / T, dim=-1).detach()
    kl = F.kl_div(student_soft, teacher_soft,
                  reduction="batchmean") * (T ** 2)
    return alpha * ce + (1 - alpha) * kl
L = α·CE(student, y) + (1−α)·T²·KL(pteacherT ‖ pstudentT)
