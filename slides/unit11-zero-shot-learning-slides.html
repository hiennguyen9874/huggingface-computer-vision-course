01 / Generalization

Nhận ra ảnh mới
chưa đủ để mở rộng thế giới

  • Train trên mèo thật → test mèo thật khác: thường ổn.
  • Train mèo thật → test mèo hoạt hình: domain shift.
  • Train cat, dog, horse → test zebra: class hoàn toàn mới.
  • Domain adaptation đổi phân phối; ZSL đổi class.

Conventional supervised

Cố định class

  • image → label
  • Classifier học trực tiếp từ ảnh đã gán nhãn.
  • Softmax chỉ biết các class được định nghĩa trước.
  • Thêm class mới thường cần train lại / fine-tune.

Zero-shot learning

Suy luận class mới

  • image + semantics → class
  • Ảnh của unseen class không xuất hiện khi train.
  • Semantic information là cầu nối giữa cái đã biết và cái mới.
  • Thêm class bằng embedding, không nhất thiết train lại.

Classical ZSL

Chỉ unseen ở test

  • Train: cat, dog, horse
  • Test: zebra, tiger, lion
  • Mô hình chỉ chọn trong Y_unseen.
  • Thiết lập sạch, cổ điển hơn.

Generalized ZSL · GZSL

Seen + unseen ở test

  • Test: cat, dog, horse, zebra, tiger, lion
  • Khó hơn vì mô hình bị kéo về seen classes.
  • Ảnh zebra dễ bị nhầm thành horse.
  • Cần calibration để cân bằng hai miền.

03 / Dữ liệu được phép dùng

Inductive hay
transductive?

  • Inductive: không thấy bất kỳ ảnh unseen nào — kể cả ảnh không nhãn.
  • Transductive: được xem thêm attributes hoặc ảnh unseen không nhãn; không có label unseen.
  • Cả hai vẫn cần attribute / text / label embedding để biết unseen class.
  • Transductive có thể tốt hơn vì tận dụng thêm cấu trúc target domain.

05 / Semantic embedding

Vector chứa
quan hệ ngữ nghĩa

  • Embedding không phải số ngẫu nhiên; các điểm gần nhau thường có nghĩa gần nhau.
  • Ví dụ kinh điển: king − man + woman ≈ queen
  • Trong ZSL, ảnh và class label cần gặp nhau trong một semantic space.
  • Nhưng semantic gần chưa chắc visual đã đủ chi tiết — đây là semantic gap.

08 / Minimal PyTorch idea

Project visual feature
vào semantic space

zsl_projection.py
class ZSLProjectionModel(nn.Module):
    def __init__(self, visual_dim, semantic_dim):
        super().__init__()
        self.projection = nn.Sequential(
            nn.Linear(visual_dim, 1024),
            nn.ReLU(),
            nn.Linear(1024, semantic_dim)
        )

    def forward(self, visual_features):
        z = self.projection(visual_features)
        return F.normalize(z, dim=-1)

Loss có thể là MSE, cosine similarity, contrastive hoặc ranking loss.

  • “a photo of a cat”

Vì sao CLIP mạnh?

Pretrain đa phương thức

  • Học từ rất nhiều cặp (image, text).
  • Dùng natural language thay cho attribute cố định.
  • Generalize ra nhiều task và open vocabulary.
  • Bản chất: multi-modal contrastive learning.

14 / What breaks?

Bốn thách thức
kỹ thuật

  • Domain shift: mapping học trên seen không còn đúng trên unseen.
  • Hubness: vài vector thành nearest neighbor của quá nhiều điểm.
  • Semantic gap: “tiger” gần “lion” bằng chữ, nhưng visual cần stripes.
  • Seen-class bias: GZSL thường kéo zebra về horse; cần calibration.
