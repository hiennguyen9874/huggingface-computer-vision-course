Sampling code

Uniform sampling: đơn giản, dễ tái sử dụng

sampling.py
def uniform_sample_frames(video, num_frames):
    # video: [T, H, W, C]
    total = video.shape[0]
    indices = torch.linspace(
        0, total - 1, steps=num_frames
    ).long()
    return video[indices]

# 100 frames → [0, 14, 28, …, 99]

RNN limitations

Vì sao RNN dần nhường chỗ?

  • Long-term dependency: video dài làm thông tin đầu chuỗi dễ bị quên.
  • Sequential: phải xử lý từng frame, khó song song hóa.
  • Slow: training và inference kém hiệu quả trên hardware hiện đại.
